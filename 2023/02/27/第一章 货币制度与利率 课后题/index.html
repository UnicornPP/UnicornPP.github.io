1.货币的本质特征

商品
具有使用价值和价值
-一般等价物
1.商品的价值体现物
2.直接同所有商品交换的能力

2.货币在商品经济中发挥哪些职能？

  • 价值尺度
    足值货币 衡量商品的价值
  • 流通手段
    非足值货币即可 强调的是变现能力 G-W-G的过程
  • 贮藏手段
    足值的金属货币 在过去具有自发调节货币流通的问题
  • 支付手段
    只交付现金，但不交换商品 比如：债券 -> 直接产生了信用货币
  • 世界货币职能
    包括国际一般支付手段、购买手段、财富转移手段

3.货币制度主要由哪些要素构成？

货币金属和货币单位、通货的铸造、发行和流通程序、金准备制度

9.现代经济中利率的杠杆作用表现在什么地方？

宏观

  • 积累资金 -> 通过利率来吸收闲置资金
  • 调整信用规模 -> 提高利率可以缩小信用规模
  • 调节国民经济结构 -> 分配不均匀的资本
  • 调节通货膨胀 从两个方面:

1.物价上升 -> 提高贷款利率 -> 货币供应量减少 -> 货币需求量减少 —> 物价回落
2.(供不应求)物价上涨 -> 降低企业短期生产利率 -> 提高产量 -> 供需平衡，物价回落

  • 平衡国际收支

微观

  • 激励企业提高资源使用效率
  • 影响家人和个人的金融资产
  • 作为租金计算的基础

10.各种利率决定理论的优缺点及比较

  1. 预期理论
    优点：解释了长期利率是由短期利率和未来预期短期利率共同决定的，因此当短期利率降低，未来预期短期利率上升，导致长期利率上升，形成了上扬的收益率曲线。
    缺点：没有解释现实中收益率曲线上扬的情况较多
  2. 市场分割理论
    优点：解释现实中收益率曲线上扬的情况较多的情况 -> 不同期限的利率水平由其供求情况决定，且大多数投资者偏好短期利率
    缺点：认为长期利率和短期利率是两个独立的市场
  3. 流动性偏好理论
    优点：收益率曲线受到预期收益率以及风险的影响，可以解释实中收益率曲线上扬的情况较多的情况
    缺点：在部分市场中，投资者并不一定偏向于短期投资，无法解释该情况
